About this Resource Page

This page is a curated collection of essential tools, documentation, and learning platforms for frontend developers. Whether you are just starting out or looking to refine your skills, you will find everything you need to build modern websites right here.


Learn and grow

  • W3Schools

    Learn coding from scratch

  • Roadmap.sh

    The place where I learn to create various projects.

  • MDN Web Docs

    Official documentation for HTML, CSS, & JavaScript.

  • freeCodeCamp

    A place to learn coding while working on real projects for free.

  • JavaScript.info

    One of the most comprehensive and easy-to-understand JavaScript learning resources on the internet.

Tools and Code Validators

  • Google Fonts

    Places to find cool fonts for the web.

  • Coolors

    A site for creating and finding beautiful color combinations with a single click.

  • Figma

    To design the website's appearance before writing the code.

  • W3C Validator

    The official site to check whether your HTML tags are written correctly and free of errors.

  • Can I Use

    A site for checking which HTML/CSS features are supported by specific browsers. For example: "Will this feature work in older versions of Chrome?"

Free Assets

  • Unsplash

    A site for high-resolution, free, and copyright-free images.

  • Placehold.co

    A site for creating placeholder or dummy images. This is incredibly useful if you don't have the actual images yet but want the website to look neat when you apply CSS later.

Code Playground

  • CodePen

    Test HTML/CSS code snippets without needing to open a code editor.

Community

  • Stack Overflow

    A place to ask questions about error codes.
